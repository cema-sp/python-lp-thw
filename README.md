# Examples from [Learn Python The Hard Way][1] book #

To run any eample use:  

```bash
python ex0NN.py
```

To source .vimrc after vim initialization:

```bash
vim -S .vimrc
```

To load .vimrc from VIM use:  

```
:so .vimrc
```

To start GNU screen use:

```bash
screen -c .screenrc
```


[1]: http://learnpythonthehardway.org/
